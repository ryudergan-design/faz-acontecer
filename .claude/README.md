# Espelho Claude

Esta pasta representa a estrutura-alvo da adaptacao do `Faz Acontecer` para o runtime `Claude Code`.

Ela existe para:
- espelhar a forma como o GSD oficial conversa com o Claude
- organizar a futura adaptacao PT-BR
- preservar a estrutura essencial esperada por esse runtime

## Objetivo

Quando a adaptacao pratica comecar, esta pasta deve receber equivalentes em portugues de:
- comandos
- agentes
- hooks
- artefatos do `get-shit-done`

Sem quebrar:
- naming estrutural
- comandos oficiais
- fluxo do GSD

## Estrutura esperada

- `commands/`
- `agents/`
- `hooks/`
- `get-shit-done/`

## Regra

Esta pasta e um espelho estrutural.
Ela nao substitui a referencia oficial.
Ela organiza a adaptacao.
