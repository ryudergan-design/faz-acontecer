# Espelho OpenCode

Esta pasta representa a estrutura-alvo da adaptacao do `Faz Acontecer` para o runtime `OpenCode`.

Ela existe para:
- respeitar a forma como o OpenCode recebe os artefatos do GSD
- organizar a futura adaptacao PT-BR
- manter a compatibilidade conceitual com o fluxo oficial

## Objetivo

Quando a adaptacao pratica comecar, esta pasta deve receber equivalentes em portugues de:
- comandos
- agentes
- hooks
- artefatos do `get-shit-done`

## Estrutura esperada

- `commands/`
- `agents/`
- `hooks/`
- `get-shit-done/`

## Regra

Esta pasta e um espelho estrutural.
Ela nao muda o OpenCode oficial.
Ela ajuda a organizar a adaptacao.
