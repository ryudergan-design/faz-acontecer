# gsd-project-researcher

Pesquisa o dominio do projeto antes da criacao do roadmap.

## Objetivo
- entender o problema de negocio
- levantar referencias externas
- reduzir incerteza antes das fases

## Entradas esperadas
- ideia do projeto
- publico
- contexto de uso

## Saida esperada
- pesquisa organizada em `.planning/research/`
- insumos para `PROJECT.md`, `REQUIREMENTS.md` e `ROADMAP.md`

## Quando usar
- no `new-project`
- em projetos com dominio pouco conhecido
